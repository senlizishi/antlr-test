#### 概述
一个使用 antlr4 生成简单四则运算语法分析器生成器的例子

#### 步骤
1、IDEA 安装 ANTLR Plugin 插件
2、Pom 文件中引入 ANTLR 依赖
3、编写 .g4 语法文件，并生成代码
4、应用
